Qué contiene

El archivo centros.csv parte del conjunto oficial de centros educativos y añade información que aparece vacía o distribuida en otras fuentes: código y nombre del Centro del Profesorado, CER, EOEP y zona de inspección. No se publica el nombre de la persona inspectora.

Los CEP, CER, EOEP y oficinas Medusa se mantienen dentro del mismo listado, identificados mediante las columnas de tipo y etapa, para evitar fragmentar la información en varios archivos.

Cómo se genera

  1. Se consulta la API CKAN de Datos Abiertos de Canarias.
  2. Se descarga el listado oficial más reciente de centros educativos.
  3. Se cruzan los códigos de centro con el conjunto oficial de zonas de inspección.
  4. Se completan las asignaciones de CEP mediante una tabla curada y versionada.
  5. Se conservan y normalizan los datos de CER y EOEP disponibles.
  6. Se generan y validan centros.csv y centros.json.

Actualización

Un workflow se ejecuta cada noche. Cuando detecta cambios en las fuentes, regenera los archivos y abre o actualiza un pull request. Ningún cambio entra directamente en la rama principal sin revisión.
